What is AntiSL?

AntiSL is one of the best anti-lag plugins that reduces lag, improves the tps and decreasing the chances for a server to crash. It provides a better server performance gives a much better experience by drastically increasing the tps of the server. AntiSL has been tested many times since its currently in a beta version. Performance

AntiSL is one of the best and easiest to use plugin that fixes most of the server lag. By installing the plugin you are guaranteed a better performance and tps without any cost.

Optimization 

AntiSL automatically improves the server's performance every 10 minutes without having you to type any command. It decreases the RAM/CPU usage without removing any entity or change the vanilla minecraft gameplay.

AntiSL fixes and patches some of the common minecraft glitches and bugs that cause most of the server lag. It also rewrites the light generation producing more fps and less lag in no time!
